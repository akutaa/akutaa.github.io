安装hexo

  • hexo docs https://hexo.io/zh-cn/docs/ or https://theme-next.js.org/docs/
  • hexo config设置域名
  • config内language设置应与theme/languages一致，如NexT简中为zh-CN而不是zh-Hans

安装NexT

  • NexT docs https://theme-next.iissnan.com/getting-started.html
  • NexT docs的仓库地址未更新，应改为 https://github.com/next-theme/hexo-theme-next
  • 给命令行添加代理
    • git: config
    • cmd: 环境变量
